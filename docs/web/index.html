Quick Start Examples

Local Testing

~1 sec
state-actor \ --db ./chaindata \ --genesis genesis.json \ --accounts 1000 \ --contracts 500 \ --seed 1

CI/CD Pipeline

~5 sec
state-actor \ --db ./chaindata \ --genesis genesis.json \ --accounts 10000 \ --contracts 5000 \ --seed 42

Mainnet-like State

~5 min
state-actor \ --db ./chaindata \ --genesis genesis.json \ --accounts 1000000 \ --contracts 500000 \ --max-slots 50000 \ --seed 12345

Erigon Format

MDBX
state-actor \ --db ./erigon-data \ --output-format erigon \ --accounts 10000 \ --contracts 5000 \ --seed 42

Target Size

5GB cap
state-actor \ --db ./chaindata \ --genesis genesis.json \ --target-size 5GB \ --accounts 10000000 \ --contracts 5000000

Binary Trie (EIP-7864)

experimental
state-actor \ --db ./chaindata \ --genesis genesis.json \ --binary-trie \ --accounts 10000 \ --seed 42

CLI Reference

Flag Default Description
--db required Output database directory
--output-format geth Output format: geth (Pebble) or erigon (MDBX)
--genesis - Path to genesis.json file (enables genesis block writing)
--accounts 1000 Number of EOA accounts to create
--contracts 100 Number of contracts to create
--max-slots 10000 Maximum storage slots per contract
--min-slots 1 Minimum storage slots per contract
--distribution power-law Storage distribution: power-law, uniform, exponential
--seed 0 Random seed (0 = current time)
--target-size - Target DB size on disk (e.g. 5GB, 500MB)
--code-size 1024 Average contract code size in bytes
--batch-size 10000 Database batch size
--workers NumCPU Number of parallel workers (0 = auto)
--binary-trie false Generate for EIP-7864 binary trie mode
--inject-accounts - Comma-separated addresses to pre-fund with ETH
--chain-id 0 Override genesis chainId (0 = use from file)
--verbose false Enable verbose output
--benchmark false Print detailed performance stats
